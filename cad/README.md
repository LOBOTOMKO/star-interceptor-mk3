# cad/

Mechanical design — modelled in Autodesk Fusion 360, printed via PrusaSlicer.

Includes dimensioned technical drawings (dome and base) with tolerances, section views and title blocks.

**STL meshes:** `stl/` holds the current printable meshes — open one on GitHub to get its interactive rotate/zoom viewer. More early-stage variants and other archived files live in the earlier [Star-Interceptor](https://github.com/LOBOTOMKO/Star-Interceptor) repository (`3D/`).

**Add here:** Fusion exports (`.f3d` / `.step`) and the technical-drawing PDFs.
